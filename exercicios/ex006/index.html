Principais formatações

Negrito/destaque

Nesta frase temos um termo em negrito usando a tag <strong>

Italico/Ênfase

Nesta frase temos um termo em italico usando a tag <em>

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Tags obsoletas

Texto grande e pequeno

estamos criando um texto grande usando a tag < big> e um texto pequeno usando a tag <small>.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>.

Texto subescrito

Para inserir coisas do tipo H2O usando tag <sub>.
